# IEEE 802.11-2007 (WLAN) Mini PHY

A basic 802.11 OFDM PHY layer,
including packet detection, synchronization, channel estimation, modulation and demodulation
using MATLAB.

> [!NOTE]
> This is a mini project of UCSD ECE257A (Fall 2024).
